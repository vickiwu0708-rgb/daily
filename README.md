# 日常 (Daily)

记录日常生活、学习和工作的笔记仓库。

A repository for recording daily life, learning, and work notes.

## 目录结构 / Structure

```
daily/
├── notes/          # 日常笔记 / Daily notes
├── learning/       # 学习记录 / Learning records
└── README.md
```

## 使用方法 / Usage

在 `notes/` 目录下按日期创建文件，例如 `notes/2026-04-24.md`，记录当天的日常内容。

Create files by date in the `notes/` directory, e.g. `notes/2026-04-24.md`, to record daily content.
